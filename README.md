# h5-nfl-archive

## 项目简介

`h5-nfl-archive` 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面资源的集合与展示用途。

项目中收录的页面均为独立 HTML 文件，无需依赖后端服务或数据库支持，可直接在浏览器中打开查看。

## 目录说明

```
h5-nfl-archive/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── index.html      # 入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：该目录下存放所有归档的 HTML 页面文件，文件按时间或主题命名。
- `assets/`：存放页面引用的公共资源文件，如 CSS、JavaScript 和图片等。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，每个页面文件可单独运行。页面内容可能涉及多种主题或用途，但不针对任何特定域名或网站进行收录。

归档的页面遵循以下原则：

- 页面之间相互独立，无交叉依赖。
- 不包含恶意代码、跟踪脚本或广告内容。
- 资源文件尽量内联或放置于 `assets/` 目录下，便于离线查看。

## 使用方式

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/your-username/h5-nfl-archive.git
   ```
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务部署整个仓库，实现线上访问。

## 维护说明

- 欢迎提交 Pull Request 添加新的归档页面，或对现有页面进行修复与优化。
- 提交的页面请确保为独立 HTML 文件，并放置在 `pages/` 目录下。
- 如有公共资源更新，请统一放入 `assets/` 目录。
- 本仓库不接受包含恶意代码、侵权内容或违反相关法律法规的页面。
